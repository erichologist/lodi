🎨 CUSTOMIZE IN 2 MINUTES:

Change Gradient Colors:
  Edit: index.html
  Look for: <circle class="gradient-node" ...>
  Change: fill="#ff006e" to any hex color

  Current colors:
  • #ff006e (Hot Pink)
  • #8338ec (Purple)
  • #3a86ff (Blue)
  • #fb5607 (Orange)
  • #06ffa5 (Mint Green)

Change 3D Object Material:
  Edit: main.js
  Find: create3DObject() function
  Modify: MeshPhysicalMaterial properties

  Properties to tweak:
  • color: hex color
  • metalness: 0 (matte) to 1 (mirror)
  • roughness: 0 (shiny) to 1 (matte)
  • emissive: glow color

Change 3D Geometry:
  In main.js, replace:
  const geometry = new THREE.IcosahedronGeometry(1, 4);

  Try:
  • TorusKnotGeometry(1, 0.4, 100, 16)
  • BoxGeometry(2, 2, 2, 32, 32, 32)
  • DodecahedronGeometry(1, 0)
  • OctahedronGeometry(1, 4)

Adjust Animation Speed:
  In main.js, modify CONFIG object:
  • rotationSpeed: increase for faster spin
  • objectScale: change object size
  • disturbanceRange: how far nodes move
  • returnSpeed: how quickly they return
